Project: Bitcoin.in.th Merchant Account
Module: Opencart Payment Module
Author: dave@advancedstyle.com

Installation:
1) Create a merchant account at http://bitcoin.in.th/merchant-account/
2) Copy all contents of the /bitcointhai_opencart/upload/ folder to your root Opencart directory
3) Go to Opencart Admin -> Extensions -> Payment -> Click "Install" next to Bitcoin Thai
4) "Edit" the Bitcoin module and enter your API ID and API Key from bitcoin.in.th
